How CareerMind Works

CareerMind is a hybrid neuro-symbolic expert system built with Python and Flask. It combines three distinct reasoning layers — classic forward-chaining rules, explicit symbolic rules with numeric thresholds, and a knowledge graph traversal — to produce ranked, explainable career recommendations. The system uses {{ rule_count }} IF-THEN rules across 20+ career domains.

📥

1. User Input

The user selects interests and skills from grouped categories, then rates subject performance using dynamic sliders that adapt to chosen interests.

🧠

2. Working Memory

Inputs are converted into a set of boolean facts. Academic scores above thresholds (e.g. math ≥ 65) trigger additional derived facts like high_math_score.

⚙️

3. Classic Forward Chaining

The engine scans all {{ rule_count }} IF-THEN rules. A rule fires when every condition is present in working memory (AND logic). Each fired rule adds its weight to that career's score.

⚡

4. Symbolic Rules

38 explicit rules blend boolean facts with numeric score thresholds (e.g. "AI interest + Math ≥ 70"). Each triggered rule contributes a boost to the career's rule score.

🕸

5. Knowledge Graph

A NetworkX directed graph connects interests, skills, subjects, and careers via weighted edges. Multi-hop traversal scores careers by the product of edge weights along each path.

🔀

6. Fusion

All three layer scores are normalised to [0, 1] and fused: Final = α × Graph + β × Rule, where α = β = 0.5. The top career always shows 100% relative confidence.

Technology Stack

  • Language: Python 3.x
  • Web Framework: Flask
  • AI Technique: Hybrid Neuro-Symbolic Expert System
  • Layer 1: {{ rule_count }} hand-crafted IF-THEN rules with forward chaining
  • Layer 2: 38 explicit symbolic rules with numeric thresholds
  • Layer 3: NetworkX directed knowledge graph (Interest → Skill → Career)
  • Fusion: Weighted score combination (α = β = 0.5)
  • UI: Pure HTML/CSS — no external JS frameworks
