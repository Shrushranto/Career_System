How CareerMind Works

CareerMind is a symbolic AI / expert system built with Python and Flask. It uses a hand-crafted knowledge base of {{ rule_count }} IF-THEN rules and a forward-chaining inference engine to reason about career suitability.

📥

1. User Input

The user selects interests, skills, and enters academic scores via the form.

🧠

2. Working Memory

Inputs are converted into boolean facts and loaded into working memory. Academic thresholds (≥65%) trigger additional derived facts.

⚙️

3. Forward Chaining

The inference engine scans all rules. A rule fires if ALL its conditions are present in working memory (AND logic).

📊

4. Scoring

Each fired rule adds its weight to the career's score. Multiple rules for the same career accumulate — this is the neuro-symbolic component.

🔍

5. Explanation

Every recommendation includes the rule IDs that fired and human-readable explanation bullets — full transparency, no black box.

🏆

6. Ranking

Careers are ranked by accumulated score. A relative confidence percentage shows how strongly each career was supported.

Technology Stack

  • Language: Python 3.x
  • Web Framework: Flask
  • AI Technique: Rule-Based Expert System with Forward Chaining
  • Knowledge Base: {{ rule_count }} hand-crafted IF-THEN rules across 20+ career domains
  • Scoring: Weighted rule accumulation (neuro-symbolic layer)
  • UI: Pure HTML/CSS with no external JS frameworks
